# 📖 About The Project

EventSphere is a next-generation immersive event management platform built with a futuristic cyberpunk-inspired interface.

The platform reimagines how conferences, hackathons, summits, and keynote events are experienced digitally through:

✨ Holographic-inspired UI  
🤖 AI-powered networking  
🎟️ Interactive ticket booking  
📊 Real-time organizer analytics  
🌌 Immersive animations & visual effects  

The goal of EventSphere is to create a cinematic and engaging event ecosystem where attendees and organizers interact inside a highly immersive digital environment.

# 🚀 Installation & Setup

## 1️⃣ Clone the Repository

```bash
git clone https://github.com/your-username/event-sphere.git
```

## 2️⃣ Navigate Into The Project Folder

```bash
cd event-sphere
```

## 3️⃣ Run The Project

Simply open:

```bash
index.html
```

in your browser.

OR use VS Code Live Server:

Right click on `index.html`  
Click `Open with Live Server`
